Commands for the terminal:

  • "cd ~" - root directory
  • "rm -fr hello" - remove directory "hello"
  • "rm -rf *" - remove everything from the folder (dangerous!)
  • "ctrl + ~" - open terminal in VS
  • "shift + command + P" - all at once to search in Visual Studio
  • "shift + 1 + enter" - html structure
  • "command + S" - save file

Here's a link to terminal commands cheatsheet.




Commands for Git:

  • "git init" - initialize git
  • "git add ." - add everything in this folder/file directory to git
  • "git commit -m "" " - second command after add to send items to git
  • "git push" - the third command in that order, one that pushes files into GitHub
  • "git status" - to check for changes in git
  • "git remote -v" - checks which depository it is connected to
  • "rm -rf .git" - removes any git depository from directory (to undo it, run git initial-scale
  • When creating new depository on GitHub, copy and paste the three lines from below the readme.md file on GitHub into the terminal directory on your PC.

General Notes and Commands:

  • C# is a Microsoft version of JAVA
  • heap memory -> execution context -> call stack
  • "Command + /" - to comment out highlighed block
  • "Shift + Opt + 'down arrow'" - make a copy of a line of code
  • "Command + F" - to change one word in whole code
  • Data Structures - list, tuples, ranges, strings
  • CRUD - stands for Create Read Update Delete
  • "passing by reference" (uses the same memory address) vs "passing by value"
  • Tuples are in parantheses, not brackets
  • when writing functions, make them descriptive
  • if you write functions and are expecting to see result but get none, check your return statement
  • "fxn()" - paranthesis invoke the fxn, it's what calls the function
  • "While" loop is used when you don't know how many iterations will be needed
  • "API" - database from outside source
  • object is a dictionary
  • "OPT + Z" - to make a long string a paragraph
  • "__init__" - a constructor, call object automatically
  • "self." - belongs to and object it's included in
  • "Command + D" - selects all instances of the word in code
  • a variable OUTside __init__ is class variable
  • a variable Inside __init__ is instance variable
  • metadata - info about the page read by servers
